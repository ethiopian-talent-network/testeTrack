### TasteTrack

TasteTrack is a mobile application developed using Flutter that enables users to discover, create, and manage recipes efficiently. The application integrates modern UI design with device capabilities such as camera and location services to enhance user experience.

---

### Project Description

TasteTrack serves as a digital platform for individuals interested in cooking and recipe management. Users can browse existing recipes, create and store their own recipes with images, and explore food options based on their current location.

The application is built with a focus on performance, scalability, and maintainability. It uses the Provider package for state management and local storage solutions such as SQFlite and SharedPreferences for data persistence.

---

### Key Features

* **Recipe Dashboard**
  Provides an organized interface for browsing recipes using a structured card-based layout.

* **Recipe Creation**
  Allows users to add new recipes with detailed ingredients and step-by-step instructions.

* **Camera Integration**
  Enables users to capture and attach images directly from their device camera.

* **Location Services**
  Utilizes GPS functionality to support location-based features.

* **Search and Favorites**
  Includes functionality to search for recipes and save preferred items for quick access.

* **Theme Management**
  Supports both light and dark modes with persistent user preferences.

* **Responsive Design**
  Ensures compatibility across different screen sizes and device orientations.
